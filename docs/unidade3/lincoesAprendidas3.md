# **Lições Aprendidas Unidade 3**

Durante o desenvolvimento do projeto da Panelas Grill, várias lições importantes foram aprendidas. Abaixo estão as lições aprendidas, focando nas ações de melhoria, desafios enfrentados e como foram (ou não) superados.

**Lições Aprendidas e Melhorias para o Processo**

<ol>
  <li>
    <strong>Desenvolvimento do MVP</strong>
    <ul>
      <li><strong>A.</strong> Desafio: Enfrentamos dificuldades para organizar o desenvolvimento e  dividir as tarefas entre a equipe,</li>
      <li><strong>B.</strong> Ação de melhoria: Buscamos compartilhar com as equipes tanto nossas facilidades quanto nossas dificuldades, a fim de garantir uma divisão justa das tarefas, sem prejudicar o andamento do projeto. Além disso, organizamos o trabalho criando branches específicas para cada etapa no GitHub, mantendo o projeto bem estruturado.</li>
    </ul>
  </li>
  <li>
    <strong>Estudos de caso</strong>
    <ul>
      <li><strong>A.</strong> Desafio: Tivemos dificuldades para compreender o PBB e o USM.
      </li>
      <li><strong>B.</strong> Ação de melhoria: Realizamos discussões em equipe e buscamos orientação da nossa monitora para executar as atividades propostas.</li>
    </ul>
  </li>
</ol>
